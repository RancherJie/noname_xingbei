## 启动方法

双击index.html即可启动游戏，网页端推荐使用Chrome系内核浏览器游玩，不推荐使用Firefox浏览器。

## 启动说明

电脑版可搭配启动器启动，手机端必须使用启动器启动；使用启动器启动，报错更详细
